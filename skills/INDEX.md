# Skills Index

Catalog of bespoke, user-defined skills available in this project. When a task matches a
skill's trigger, read that skill's `SKILL.md` and follow it.

Each skill is a self-contained folder: `skills/<id>/SKILL.md` (plus optional
`references/`). See [`../CLAUDE.md`](../CLAUDE.md) § Skills for the authoring guide and
[`_template/SKILL.md`](_template/SKILL.md) for the template.
